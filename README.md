# Setup

install the packages with your package manager, if you use node

```terminal
npm install
```
Fill env

```terminal
DATABASE_URL="dburl"

APP_PORT="8080"
```
migrate

```terminal
npx prisma migrate dev
```
Run the project, if you use node

```terminal
npm run dev
```
